# api-mashine
Laravel &lt;-> JSON API &lt;--> WEB APP

Install and run:

```js
composer install
cp .env.example .env
php artisan passport:install
php artisan migrate
php artisan db:seed
php artisan key:generate
php artisan serve
npm run dev
```

Update:

```
  npm remove laravel-mix

```


For windows host - frontend part->

```js
npm install --global gulp-cli
```
